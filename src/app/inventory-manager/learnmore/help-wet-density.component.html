  • Measuring Wet Density

This tool relies on an accurate wet density for the feed being stored. The expected wet density range for each feed type is provided as a guide only.

Measure the wet density of your feed as follows:

Silages

Cut a square or rectangle out of the stack of known dimensions (e.g. 0.2m long X 0.2m wide X 0.2m high) and weigh it using a set of accurate scales.

Wet density (kg/cubic metre) = Wet weight (kg) divided by Volume (length X width X height in metres)

Grains and concentrates, By-products

Fill a one litre container (e.g. milk carton) and weigh it using a set of accurate scales.

Wet density (kg/cubic metre) = Wet weight (kg) multiplied by 1000
